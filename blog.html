Why did this development stop

April 15th, 2018

I'm back, and it's been at least a few months since I did a tiny update on myself. If you've noticed in the projects tab, I actually added and new 'In Progress' project. I've been recruited for a very simple job, to write data for puzzles and perhaps even write functions. Honestly, I was sort of excited going into it because it seemed like a coding task and that I would learn something from this. But all I've been doing is designing puzzles, putting it into the data file or the beta file for testing, then pushing it, and waiting. I'm pretty grateful that Ken gave me a task but I'm also upset with the fact I'm not really getting much out of this but some insight to the code (which I've definitely learned quite a bit through). I had the option to say no, but I didn't because, I guess I expected more.

After this task is done, I need to find my motivation to code again. Whether it is to upgrade and update this site, or to design a game and create it, something has to happen. I'm not progressing with my free time, other than making money off my full-time job in the food industry. That's not how I want to spend this opportunity.

I hope I'll be continuing to update this site, or at least updating this blog to keep tabs and talk about my life so I can reflect on it sometime later.
